# Obsidian Canvas Remove and Delete Files

This is a simple plugin for [Obsidian](https://obsidian.md)'s Canvas that allows removing files from a canvas and automatically delete them from the vault as well.

It adds the command `Canvas Remove And Delete Files: Invoke` and you can configure your own hotkey in the Obsidian keyboard shortcut settings.

https://github.com/user-attachments/assets/8d153020-5dc5-4ac9-841a-3da284d1f66a

## Manually installing the plugin
- Head over to [the latest release](https://github.com/jffaust/obsidian-canvas-remove-and-delete-files/releases/latest) and download `main.js` and `manifest.json`
- Copy those files to your vault `VaultFolder/.obsidian/plugins/canvas-remove-and-delete-files/`.
